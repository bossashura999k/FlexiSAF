# FlexiSAF Backend Internship — Term 2, 2026

Repo for my journey through the **FlexiSAF Internship Program (FIP)** — Backend
pathway. Each `TASK` folder is a weekly deliverable, written in **Java**, covering
a new core concept of the language and Object-Oriented Programming (OOP).

> 👨‍💻 Coming from a JS/Git/Bash background, so a lot of this is "translating"
> familiar concepts (classes, loops, functions) into Java's stricter, typed world.
